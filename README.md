En este proyecto se realizó un análisis de mercado financiero utilizado SQL y Power BI, con el objetivo de evaluar la relación entre la volatilidad (riesgo) y la liquidez (volumen) de distintos activos.

este análisis permitio identificar patrones de comportamiento en el mercado y detectar activos con diferentes perfiles de riesgo y estabilidad.

volatilidad: medida como el promedio del cambio absoluto en el mercado
liquidez: medida como el volumen total de transacciones

se logro identificar que:
goog presenta la mayor liquidez, indicando una alta actividad de mercado
msdt muestra la mayor volatilidad, reflejando mayores fluctuaciones en su precios
tsla destaca al tener un gran equilibrio entre liquidez y baja volatilidad 
no se a logrado observar una relacion directa entre precio y volumen, lo que sugiere que otros factores influyen en el comportamiento del mercado 

Para una mayor presentación se desarrollo un dashboard en Power BI que incluye:
KPIs de volumen total y volatilidad promedio
Gráfico de dispersión (scatter plot) para analizar la relación entre riesgos y liquidez 

el grafico logro identificar activos con diferentes perfiles:
Alta liquidez+ baja volatilidad = activos más estables
Alta volatilidad + baja liquidez = activos más riesgosos 

este análisis logro demostrar que la liquidez y la volatilidad no están directamente relacionadas, lo que permite identificar 
oportunidades dependiendo del perfil de riesgo del inversionista.






activos como tsla puede representar un mejor equilibrio, mientras que activos como goog destacan por su alta liquidez 

Este proyecto demuestra cómo el análisis de datos puede ser utilizado para comprender el comportamiendo del mercado financiero y apoyar la toma de decisiones estratégicas basadas en datos.
